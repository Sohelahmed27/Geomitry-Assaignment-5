What is callback function? Why will you use it?

A callback function is a function passed into another function as an argument, which is then executed inside the outer function to complete some kind of action.

Callbacks allow to continue executing code while the operation is being executed in the background. Once the operation has completed, the callback function is called with the result of the operation. This way, we can ensure that the program remains responsive and the user experience is not impacted.
